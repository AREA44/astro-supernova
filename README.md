# Supernova

[![Built with Starlight](https://astro.badg.es/v2/built-with-starlight/tiny.svg)](https://starlight.astro.build)

## Getting Started

```sh
git clone https://github.com/AREA44/astro-supernova
cd astro-supernova
pnpm install
pnpm dev
```
