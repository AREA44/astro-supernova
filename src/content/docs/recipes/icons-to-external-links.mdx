This guide shows how to add icons to the end of each external link, so that visitors will know they are leaving your site.

## Step-by-Step Guide

<Steps>

1. Install the [`rehype-external-links`](https://github.com/rehypejs/rehype-external-links) package as a dependency:

   <PackageManagerCommand command="add rehype-external-links" />

2. Import the plugin and configure it in your `astro.config.mjs` file:

   ```js ins={2, 6-13}
   // astro.config.mjs
   import rehypeExternalLinks from "rehype-external-links";

   export default defineConfig({
     markdown: {
       rehypePlugins: [
         [
           rehypeExternalLinks,
           {
             content: { type: "text", value: " ↗" },
           },
         ],
       ],
     },
   });
   ```

   **Optional: Add `target` and `rel` attributes**  
   You can configure external links to open in a new tab with security attributes:

   ```js
   export default defineConfig({
     markdown: {
       rehypePlugins: [
         [
           rehypeExternalLinks,
           {
             target: "_blank",
             rel: ["nofollow", "noopener"],
             content: { type: "text", value: " ↗" },
           },
         ],
       ],
     },
   });
   ```

</Steps>

## Example

Create a sample file like `src/content/docs/example.mdx` and include the following:

```mdx
[Google](https://google.com)
```

This will generate the following HTML output:

[Google](https://google.com)
