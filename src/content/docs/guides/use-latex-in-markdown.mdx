This guide lead you use LaTeX in Starlight projects.

Start by installing the [`remark-math`](https://github.com/remarkjs/remark-math) and [`rehype-katex`](https://github.com/remarkjs/remark-math/tree/main/packages/rehype-katex) package using your favorite package manager:

<Tabs>
  <TabItem label="npm">
    ```sh
      npm install remark-math rehype-katex
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```sh
      pnpm install remark-math rehype-katex
    ```
  </TabItem>
  <TabItem label="yarn">
    ```sh
      yarn install remark-math rehype-katex
    ```
  </TabItem>
</Tabs>

Create `katex.min.css` in `styles` folder. After that copy all KaTeX fonts into `fonts/styles` folder.

Add `remark-math` and `rehype-katex` in Astro config:

```ts
...
import { defineConfig } from 'astro/config'
import starlight from '@astrojs/starlight'
import remarkMath from 'remark-math'
import rehypeKatex from 'rehype-katex'

export default defineConfig({
  ...
  markdown: {
    remarkPlugins: [remarkMath],
    rehypePlugins: [rehypeKatex],
  },
  integrations: [
    starlight({
      customCss: [..., './src/styles/katex.min.css'],
    ...
    })
  ],
})
```

That's all. Now you can use LaTeX in `.md` and `.mdx` file like this:

```md
When $a \ne 0$, there are two solutions to $(ax^2 + bx + c = 0)$ and they are
$$ x = {-b \pm \sqrt{b^2-4ac} \over 2a} $$
```

The code above generates the following on the page:

When $a \ne 0$, there are two solutions to $(ax^2 + bx + c = 0)$ and they are
$$ x = {-b \pm \sqrt{b^2-4ac} \over 2a} $$
