This guide lead you use LaTeX in Starlight projects.

## Steps by steps

<Steps>
1.  Start by installing the [`remark-math`](https://github.com/remarkjs/remark-math) and [`rehype-katex`](https://github.com/remarkjs/remark-math/tree/main/packages/rehype-katex) package using your favorite package manager:
    <Tabs>
      <TabItem label="npm">
        ```sh
        npm add katex rehype-katex remark-math
        ```
      </TabItem>
      <TabItem label="pnpm">
        ```sh
        pnpm add katex rehype-katex remark-math
        ```
      </TabItem>
      <TabItem label="Yarn">
        ```sh
        yarn add katex rehype-katex remark-math
        ```
      </TabItem>
      <TabItem label="Bun">
        ```sh
        bun add katex rehype-katex remark-math
        ```
      </TabItem>
    </Tabs>

2.  Add `katex`, `rehype-katex` and `remark-math` in `astro.config.mjs`:

        ```diff js
        // astro.config.mjs
        import { defineConfig } from 'astro/config'
        import starlight from '@astrojs/starlight'
        +import remarkMath from 'remark-math'
        +import rehypeKatex from 'rehype-katex'

        export default defineConfig({
          markdown: {
            remarkPlugins: [remarkMath],
            rehypePlugins: [rehypeKatex],
          },
          integrations: [
            starlight({
        +      customCss: [..., 'katex/dist/katex.min.css'],
            })
          ],
        })
        ```

    </Steps>

## Examples

Now you can use LaTeX in `.md` and `.mdx` file like this:

```md
When $a \ne 0$, there are two solutions to $(ax^2 + bx + c = 0)$ and they are
$$ x = {-b \pm \sqrt{b^2-4ac} \over 2a} $$
```

The code above generates the following on the page:

When $a \ne 0$, there are two solutions to $(ax^2 + bx + c = 0)$ and they are
$$ x = {-b \pm \sqrt{b^2-4ac} \over 2a} $$
